## 栈   
#### 1 基本概念  
栈是一种限定仅在表尾进行插入和删除操作的线性表。对于栈来说，表尾又称为栈顶。因为栈是一种仅在栈顶进行操作的线性表，所以又称为“后进先出”线性表。   
#### 2 实现  
##### 2.1 基于顺序存储实现的栈    
```java  
package mstack;

import java.util.Iterator;

/**
 * Created by ljf-梁燕双栖 on 2016/3/27.
 */
public class ArrayStack<T> implements Iterable<T> {

    private int N = 0;
    private static final int CAPACITY = 10;
    private T[] elements = null;

    public ArrayStack() {
        elements = (T[]) new Object[CAPACITY];
    }

    public boolean isEmpty() {
        return N == 0;
    }

    public int size() {
        return N;
    }

    /**
     * 动态扩充栈的容量
     * @param capacity ： 新的容量
     */
    private void resize(int capacity) {
        T[] temp = (T[]) new Object[capacity];
        for (int i = 0; i<N; i++) {
            temp[i] = elements[i];
        }
        elements = temp;
    }

    public void push(T element) {
        if (N == elements.length) resize(2*elements.length);
        elements[N++] = element;
    }

    public T pop() {
        T temp = elements[--N];
        elements[N] = null;
        //避免对象游离
        if (N > 0 && N ==elements.length/4) resize(elements.length/2);
        return temp;
    }

    @Override
    public Iterator<T> iterator() {
        return new MIterator();
    }
    private class MIterator implements Iterator<T> {

        private int i = N;
        @Override
        public boolean hasNext() {
            return i>0;
        }

        @Override
        public T next() {
            return elements[--i];//此方法只是获取元素的值，并没有从栈中弹出
        }

        @Override
        public void remove() {

        }
    }
}  
```  
在这种实现方式中，初始化的时候给栈分配一个容量为10的空间，等栈满后再动态分配空间。当栈中的实际元素只是栈容量的1/4时，需要收回一些空间。  

##### 2.2 基于链式存储实现的栈  
```java  
package mstack;

import java.util.Iterator;

/**Description : 基于链式存储实现的栈
 * Created by ljf-梁燕双栖 on 2016/3/27.
 */
public class LinkStack<T> implements Iterable<T>{
    private int N = 0;

    private Node head,tail;

    private class Node {
        T value;
        Node previous;
    }

    public LinkStack() {
        head = tail = null;
    }

    public boolean isEmpty() {
        return N == 0;
    }

    public int size() {
        return N;
    }

    public void push(T element) {
        if (isEmpty()) {
            head = new Node();
            tail = new Node();
            head.previous = null;
            head.value = element;
            tail.previous = head;
            tail.value = null;
            ++N;
            return;
        }
        Node node = new Node();
        node.value = element;
        node.previous = tail.previous;
        tail.previous = node;
        ++N;
    }

    public T pop() {
        T temp = tail.previous.value;
        Node p = tail.previous;
        tail.previous = null;
        tail.previous = p.previous;
        p = null;
        --N;
        return temp;
    }

    @Override
    public Iterator<T> iterator() {
        return new MIterator();
    }

    private class MIterator implements Iterator<T> {

        Node p = tail.previous;
        @Override
        public boolean hasNext() {
            return p != null;
        }

        @Override
        public T next() {
            T temp = p.value;
            p = p.previous;
            return temp;
        }

        @Override
        public void remove() {

        }
    }
}
```  
正如在前面线性表的实现中提到到一样，前面的实现是有一定的缺陷的。因此，在实现栈时进行了改进，即结点中存储元素的值和前驱结点，这样，当往表尾插入元素时不需要移动指针，直接修改指针指向即可，因此可以提高效率。  
#### 3 栈的应用  
##### 3.1 数制转换  
将十进制的整数转换成其他进制的整数  
```java    
package test;

import java.util.Stack;

/**Description : 进制转换
 * Created by ljf-梁燕双栖 on 2016/3/28.
 */
public class Conversion {

    /**
     * 将十进制的整数 n 转换为 d 进制的整数
     * @param n : 十进制数
     * @param d : 目标进制
     * @return : 转换后的结果
     */
    public static int conversion(int n, int d) {
        int N = n;
        StringBuffer s = new StringBuffer();
        Stack stack = new Stack();
        while (N > 0) {
            stack.push(N % d);
            N /= d;
        }
        while (!stack.isEmpty()) {
            s.append(stack.pop());
        }
        return Integer.parseInt(s.toString());
    }
    public static void main(String[] args) {
        System.out.println(conversion(1348, 16));
    }
}
```  
##### 3.2 括号匹配  
假设表达式中允许包含两种括号：圆括号和方括号，其嵌套的顺序随意，即([]())或[([][])]等为正确的格式，[(])或([())均为不正确的格式。  
```java  
package test;

import java.util.Stack;

/**Description : 利用栈实现括号匹配问题
 * Created by ljf-梁燕双栖 on 2016/3/28.
 */
public class BracketsMatch {

    public static void match(String s) {
        Stack stack = new Stack();
        for (int i = 0; i < s.length(); i++) {
            if (!stack.isEmpty()) {
                if ((stack.peek() == '(' && s.charAt(i) == ')') ||
                        (stack.peek() == '[' && s.charAt(i) == ']'))
                    stack.pop();
                else stack.push(s.charAt(i));
            } else {
                stack.push(s.charAt(i));
            }
        }
        if (stack.isEmpty())
            System.out.println("YES");
        else System.out.println("NO");
    }

    public static void main(String[] args) {
        String s = "[([]])]";
        match(s);
    }
}
```
